# Blogify

A modern blogging platform built with React, Redux Toolkit, Appwrite, and TinyMCE.

## Features
- User Authentication
- Create, Edit & Delete Posts
- Rich Text Editor
- Image Uploads
- Responsive Design
- State Management with Redux Toolkit

## Tech Stack
- React
- Redux Toolkit
- Appwrite
- TinyMCE
- Tailwind CSS
- Vite
